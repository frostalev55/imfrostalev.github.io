دليل مفصل لتثبيت Docker وبناء وتشغيل ملف Docker لاستخدام Apache Spark™

الجزء الأول: تثبيت Docker

باعتبار أنّا جميعا نستخدم Windows:

  1. نزل [Docker Desktop](Windows | Docker Docs) من الموقع الرسمي.
  2. تأكد من وجود WSL [Windows Subsystem for Linux] على جهازك:
    1. افتح الـTerminal\CMD\PowerShell
    2. أدخل الأمر التالي: wsl
    3. إن كان الرد كالتالي فتابع من خطوة تثبيت Docker
    4. تأكد من تفعيل WSL 2 بالخطوات التالية:
      1. ابحث عن [turn windows features on or off]
      2. تأكد من تفعيل الخيار التالي:
      3. بعدها قم بإعادة تشغيل الكمبيوتر
  3. شغل ملف التثبيت الذي نزلته:
  4. اغلق نفذة البرنامج التالية:
  5. افتح VS Code وأنشئ فولدر جديد:
  6. أنشئ ملفًا وسمه [requirements.txt] وانقل إليه النص التالي:
   pyspark==3.5.0
   jupyter==1.0.0
   notebook==7.0.6
   pandas>=2.1.0
   numpy>=1.24.0
   matplotlib>=3.7.0
   seaborn>=0.12.0
  1. أنشئ ملفًا وسمه [dockerfile] وانقل إليه النص التالي:
# Use Python 3.9 as base image
FROM python:3.9-slim

# Install Java and required tools
RUN apt-get update && \
    apt-get install -y --no-install-recommends \
        openjdk-17-jdk-headless \
        wget \
        procps \
    && apt-get clean && rm -rf /var/lib/apt/lists/*
    
# Set environment variables for Java and PySpark
ENV JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
ENV PATH=$JAVA_HOME/bin:$PATH
ENV PYSPARK_PYTHON=python3
ENV PYSPARK_DRIVER_PYTHON=jupyter
ENV PYSPARK_DRIVER_PYTHON_OPTS="notebook --ip=0.0.0.0 --port=8888 --no-browser --allow-root"

# Install Python packages
WORKDIR /workspace
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Verify Java installation and environment variables
RUN java -version && \
    echo "JAVA_HOME=$JAVA_HOME" && \
    echo "PATH=$PATH"

# Expose Jupyter port
EXPOSE 8888

# Start Jupyter Notebook
CMD ["jupyter", "notebook", "--ip=0.0.0.0", "--port=8888", "--no-browser", "--allow-root"]


8. افتح الـTerminal بداخل VS Code [CTRL + J]
9. أدخل الأمر التالي وانتظر انتهاء بناء ملف الـDocker:

docker build -t pyspark .


10. أدخل الأمر التالي وانتظر انتهاء تشغيل الـContainer:

docker run -it -p 8888:8888 -v ${PWD}:/workspace pyspark


11. ستجد رابطا في النهاية بادئته:

http://127.0.0.1:8888/?token=
  1. انسخ الرابط وافتحه في متصفح الويب على جهازك.
  2. ستظهر لك نافذة Jupyter عادية يمكنك التفاعل معها كالعادة:
  3. تأكد من أن كل شيء يعمل بشكل صحيح من خلال إنشاء Notebook جديدة وتجربة الأوامر التالية فيها:
from pyspark.sql import SparkSession

spark = SparkSession.builder \
	.appName("PySpark Test") \ 
	.getOrCreate()

data = [("Alice", 25), ("Bob", 30), ("Charlie", 35)] 
df = spark.createDataFrame(data, ["Name", "Age"])

df.show()
