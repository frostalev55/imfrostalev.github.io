Admin Functions:

Series Related Functions:

$series {subcommand}

  1. add -> use this subcommand to add a new series to the database.
    $series add {#channel-mention} (TL) (PR) (CL) (TS) (QC) (SR)
    example: $series add #Solo-Leveling 4 2 1.5 1.5 2

  2. edit -> use this subcommand to edit the points of an existing series in the database.
    $series edit {#channel-mention} (TL) (PR) (CL) (TS) (QC) (SR)
    example: `$series edit #Solo-Leveling 5 3 2 2 2

  3. staff -> use this subcommand to assign members to a series & if members were already assigned to a series, it will update them.
    $series staff {#channel-mention} (@TL) (@PR) (@CL) (@TS) (@QC) example: $series staff #Solo-Leveling @user1 @user2 @user3 @user3 @user4

  4. delete -> use this subcommand to delete an existing series from the database.
    $series delete {#channel-mention} example: $series dedlete #Solo-Leveling

  5. substitute -> use this command to request temporary/permanent staff to work on a chapter/s in a certain series.
    Usage:
    Submit a request in the #substitutes channel with the following format:
    {#channel-mention} (staff_role/s) {chapters_count} {temp/perma}
    example: #Solo-Leveling Translator

    jobs:
    Translation | Proofreading | Cleaning | Typesetting | Quality Checking

    Requirements:

    • The submission must be in the #substitutes channel.
    • Invoker must have Manager role.

Users Related Functions:

$user/u {subcommand}

  1. add/a -> use this subcommand to add a new member to the database.
    $user add {@member}
    example: $user add @user
    example: $u add @user

  2. reward/r -> use this subcommand to reward a member by adding points.
    $user reward {@member} (reward) "reason"
    example: $user reward @user 2
    example: $u r @user 2

  3. punish/p -> use this subcommand to punish a member by subtracting points.
    $user punish {@member} (reward) "reason"
    example: $user punish @user 2
    example: $u p @user 2

  4. $leaderboard/lb -> use this command to show the month's Leaderboard.
    $leaderboard {number of desired ranks}
    $leaderboard 3
    $leaderboard
    $lb
    example: $lb

  5. $point_history/ph -> use this command to show a list of the reward/punish history of a user.
    $point_history {@user}
    $ph @user ph`

  6. $reset_points -> use this command to reset all users' points and submissions.

    • the bot will send a CSV file with the points before the reset.
    • the bot will also show a list of the users and their total points.
    • only users with the role Administration can use this command.
      example: $reset_points

Progress Bot

  1. Record your progress
    #channel-name / chapter-number/s / step / comments, links, mentions, etc...
    Steps are: raw, tl, pr, cl, ts, qc

  2. %undo/u -> unregisters a chapter if sent by mistake.
    #channel-name / chapter-number / step

  3. %prog/p -> Show the progress report for a series.
    you can skip the channel mention to get the progress of the current channel.

  4. %overview/o -> Show the progress report for all series in a category. (Managers only)

    Usage:
    /mentions {include_role/s} {exclude_role/s} {hiatus_status}

    Basic Solution 2: ping the @KTL role in the #Solo-Leveling channel and ask who's the current translator.
